# Agrinho 2026 - Uso Consciente da Água na Agricultura

## Tema Oficial
**Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente**

## Assunto Escolhido
**Uso consciente da água na agricultura**

## Sobre o Projeto
Este projeto foi desenvolvido para o **Concurso Agrinho 2026**, na categoria **Programação Web**.  
A aplicação apresenta informações sobre a importância do uso responsável da água na agricultura, destacando o equilíbrio entre produção agrícola forte e a preservação do meio ambiente.

O site contém seções explicativas, boas práticas e uma **calculadora interativa de economia de água**, desenvolvida em JavaScript.

## Objetivo
Demonstrar como pequenas mudanças nas práticas agrícolas, aliadas à tecnologia, podem reduzir o consumo de água, aumentar a eficiência da produção e contribuir para um futuro mais sustentável.

## Tecnologias Utilizadas
- HTML5
- CSS3 (com design responsivo)
- JavaScript (interatividade)
- Apoio de Inteligência Artificial (para geração inicial de código)

## Funcionalidades
- Página informativa e educativa sobre o tema
- Design responsivo (funciona bem em celular, tablet e computador)
- Calculadora interativa de economia de água
- Layout clean com cores inspiradas no campo (verde e tons naturais)
